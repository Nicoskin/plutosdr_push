Задание №2 Создание графиков с различным стилем отображения

    1. В виде отсчетов (используя функцию `stem`)
    2. В виде квантованных уровней (функция `step`)
3. Дать название каждому графику:
    1. “Аналоговый”
    2. “Дискретных”
    3. “Квантованный”

5. Подписать название осей.
